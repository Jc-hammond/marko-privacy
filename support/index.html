Marko Support

Support information for Marko, a Markdown viewer and editor for macOS.

Getting Help

For Marko support, contact Connor Hammond through the Marko support issue tracker.

Before You Report an Issue

Include your macOS version, Marko version, what you were trying to do, and whether the issue happens with one Markdown file or every file.

Privacy

Marko is fully offline and collects no data. Read the privacy policy.
